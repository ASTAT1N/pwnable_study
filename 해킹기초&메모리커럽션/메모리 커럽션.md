# 초기화되지 않은 메모리

> 변수를 초기화 하지 않을때 생성되는 취약점. 공격자가 초기화 되지 않은 메모리를 조작하고 이를 초기화되었다고 가정하고 사용하면 보안 취약점이 될 수 있다.

# 정수범위

> 정수의 형 변환을 제대로 고려하지 못할때 발생하는 것.
>
> 묵시적 형 변환: 연산시 자료형이 서로 다를경우 직접 명시하지 않으면, 자동으로 형 변환이 일어나는것. 취약점으로 이어질 수 있음.
